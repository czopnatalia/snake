# How to Run / Installation
To run this project locally, follow these steps:

## 1. Prerequisites
Ensure you have Visual Studio (2019 or newer) installed with the appropriate workload (e.g., .NET desktop development or C++ development).

Make sure you have Git installed on your system.

## 2. Clone the Repository
Open your terminal or Git Bash and run:

git clone https://github.com/czopnatalia/snake

## 3. Open the Project
Navigate to the project folder.

Double-click the .sln (Solution) file to open it in Visual Studio.

## 4. Build and Run
Once the project loads, wait for Visual Studio to restore any necessary dependencies or NuGet packages.

Press F5 or click the Start button (Green Arrow) on the top toolbar to compile and run the application.
